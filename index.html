Projects

TravelBudgetFX

AIM Club Project | 2025

TravelBudgetFX is an AI-powered smart travel budgeting and itinerary planning platform designed to simplify trip planning and reduce overspending. The application combines real-time currency data, budget estimation, and personalized itineraries into a single unified experience I worked on designing and implementing a Temporal Fusion Transformer (TFT) deep learning model to forecast foreign exchange (FX) trends, enabling more accurate budget predictions for international travel. The model captures both short-term and long-term temporal patterns in currency data, helping users plan trips with better financial confidence.

Stock Price Forecast Project

Personal Project | 2025

Developed a full-stack web application that forecasts stock prices using an XGBoost Regressor machine learning model. Implemented a responsive interface allowing users to search stock symbols and view real-time prices, historical data, and 7-day forecasts. Built the backend with FastAPI to integrate financial data from Yahoo Finance and Finnhub APIs, while storing forecast results in a Supabase database. Designed the frontend with HTML, CSS, and JavaScript, including auto-suggestions and recently viewed stocks. This project demonstrates expertise in data acquisition, machine learning integration, and interactive web development.
